# 💰 FinSight - Personal Finance Management System

FinSight is a web-based personal finance management system that helps users efficiently manage their income, expenses, and personal budgets. The application provides a simple and user-friendly interface for tracking financial records and monitoring spending habits.

---

## 📌 Features

- 👤 User Registration & Secure Login
- 🔒 Forgot Password with OTP Verification
- 📊 Dashboard with Financial Overview
- 💵 Add Income Records
- 💸 Add Expense Records
- 🗑️ Delete Income & Expense Entries
- 👤 User Profile Management
- 📱 Responsive User Interface

---

## 🚀 How to Run

1. Install **XAMPP**.
2. Start **Apache** and **MySQL**.
3. Copy the project folder into:

```
xampp/htdocs/
```

4. Import the SQL database into **phpMyAdmin**.
5. Update the database configuration in `config.php`.
6. Open your browser and visit:

```
http://localhost/FinSight
```
